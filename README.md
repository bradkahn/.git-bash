# Git Bash

Add Git completion and prompt to your bash shell.

## Instructions

Clone this repo into your home directory:

`cd ~; git clone https://github.com/bradkahn/git-bash.git`

Run the init script:

`cd git-bash; source ./git-bash.sh`
